# Real State

Full stack React, Node.js Real Estate App tutorial with MongoDB and real-time chat functionality with Socket.io. Mern Stack project using Prisma, JWT, Cookies, Context API, React Router Dom, and more..

## Installation

Install my-project with yarn

```bash
  yarn add 
  cd api
  cd client
  cd socket
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/mortiff75/real_state.git
```

Go to the project directory

```bash
  cd clinet / api / socket
```

Install dependencies

```bash
  yarn add
```

Start the api/app.js

```bash
for each side:
  npm run dev
```
